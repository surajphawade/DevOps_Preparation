# Azure Cloud & Security – EPAM

## Q1. How do you secure Azure environment end-to-end?
### Answer
Use RBAC, NSG rules, Azure Firewall, Private Endpoints, Managed Identities, Key Vault, Defender for Cloud, and policy enforcement.
### Follow-Up Points
- Disable public access  
- Enforce governance with Azure Policy  
### Example
App → Managed Identity → Key Vault → Private Endpoint.
