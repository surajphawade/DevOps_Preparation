## Q2. What is a production cluster?
### Answer
A high-reliability cluster with autoscaling, ACLs, and stable compute for ETL workloads.
### Follow-Up Points
- Job-only  
- Spot instances optional  
### Example
Production ETL cluster for dashboards.

---

## Q3. Difference between ADF Data Flow and Databricks?
### Answer
ADF → ETL orchestration  
Databricks → Spark-based transformation engine.
### Follow-Up Points
- Databricks is compute  
### Example
ADF triggers Databricks job.
