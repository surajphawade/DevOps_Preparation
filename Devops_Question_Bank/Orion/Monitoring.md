# Monitoring – Orion

## Q1. How do you monitor Kubernetes cluster?
### Answer
Azure Monitor, Container Insights, Prometheus/Grafana.
### Follow-Up Points
- Track node health, pod metrics  
### Example
Enable Insights → Log Analytics workspace.

---

## Q2. How do you monitor pods?
### Answer
Use kubectl commands, metrics server, Container Insights, custom dashboards.
### Follow-Up Points
- Check CPU/memory  
### Example
kubectl top pod
